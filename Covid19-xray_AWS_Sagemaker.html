Prepare the dataset ¶

This dataset contains X-Ray images of both healthly patients and patients with Pneumonia (COVID-19) affected patients along with a few other diseases such as SARS and ARDS.

The images are stored on GitHub here: https://github.com/ieee8023/covid-chestxray-dataset/tree/master/images

To download the data to notebook's EBS volume, run the command "!git clone https://github.com/ieee8023/covid-chestxray-dataset". The path to images will be SageMaker/covid-chestxray-dataset/images.

The offet is the number of days since EITHER 1.) the start of the patient's symptoms OR 2.) the number of days since the hospitalization of the patient. This field will be important, especially when there are multiple X-Rays for the same patient across the duration of the patient's hospital stay. The condition of the lungs will likely change during the hospital stay.

I do not understand what the negative values in this column represent. Maybe patients that did not have symptoms at the time of X-Ray and developed symptoms later?

Hyperparameter Tuning Job ¶

  • Define which parameters to tune, along with bounds of possible values to choose from.
  • Specify the objective is to minimize validation loss
In [62]:
from sagemaker.tuner import IntegerParameter, CategoricalParameter, ContinuousParameter, HyperparameterTuner

hyperparameter_ranges = {
    'learning_rate': ContinuousParameter(0.001, 0.2, scaling_type='Logarithmic'),
    'epochs': IntegerParameter(16, 64),
    'batch_size': IntegerParameter(4, 16)
}

metric_definitions = [{'Name': 'loss',
                       'Regex': ' loss: ([0-9\\.]+)'},
                     {'Name': 'val_loss',
                       'Regex': ' val_loss: ([0-9\\.]+)'}]

objective_metric_name = 'val_loss'
objective_type = 'Minimize'
